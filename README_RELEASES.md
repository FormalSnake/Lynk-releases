# Lynk Browser Releases

This repository hosts official releases for [Lynk Browser](https://github.com/formalsnake/lynk-browser).

## Download

Download the latest version for your platform from the [Releases](https://github.com/formalsnake/Lynk-releases/releases) page.

| Platform | Format |
|----------|--------|
| macOS | `.dmg` |
| Windows | `.exe` (installer) |
| Linux | `.AppImage`, `.deb`, `.snap`, `.flatpak` |

## Auto-Updates

Lynk Browser automatically checks for updates from this repository. When a new version is available, you'll be notified within the app.

## Verifying Downloads

All macOS releases are signed and notarized by Apple.
